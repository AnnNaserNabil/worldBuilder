# World Builder Engine

> An AI-generated dark fantasy realm with interactive visualizations

## 🌍 Overview

The World Builder Engine is a complete autonomous world-building system that generates a fictional dark fantasy world with:

- **Interactive World Map** - Explore regions, biomes, and cities with zoom/pan capabilities
- **Historical Timeline** - Journey through epochs of history with significant events
- **Character Network** - Visualize relationships between characters using force-directed graphs
- **Faction System** - Discover the powers that shape the world
- **Economy Dashboard** - Track resources, trade routes, and market trends
- **Lore Library** - Read creation myths, magic systems, and geographical information

## 🚀 Quick Start

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

## 🤖 Agent System

The world is generated by 5 specialized AI agents:

1. **LORE_SMITH** - Creates the world bible including overview, creation myth, geography, magic system, and factions
2. **CARTOGRAPHER** - Generates map data and creates the interactive map component
3. **CHRONICLER** - Creates timeline epochs and historical events
4. **CHARACTER_WEAVER** - Generates characters with complex relationship networks
5. **ECONOMY_ORACLE** - Simulates economy with resources, trade routes, and markets

To regenerate world data:
```bash
node agents/loreSmith.js
node agents/cartographer.js
node agents/chronicler.js
node agents/characterWeaver.js
node agents/economyOracle.js
```

## 🔧 Development

### Adding New Pages
1. Create component in `src/pages/`
2. Add route in `src/App.tsx`
3. Add navigation link in `src/components/Layout.tsx`

### Adding New Data Types
1. Define TypeScript interface in `src/types/world.ts`
2. Update agent to generate data
3. Add to Zustand store

### Customizing Theme
Edit `src/index.css` to modify:
- Color variables in `@theme`
- Animation keyframes
- Utility classes
